Getting Started

To use the AI Agent Host, follow these steps:

  1. Write the image file onto a 64GB SD Card .
  2. Insert the preloaded micro SD card into your Raspberry Pi and boot.
  3. Open an SSH session with the default user: dietpi and change directory with the command line:

    4. cd Raspberry-Pi-AI-Agent-Host

  4. Start Services from the command line:

    5. sudo docker compose up --build -d

  5. After Usage Stop Services from the command line:

    6. sudo docker compose -v down

Usage

Once the services are up and running, you can access the AI Agent Host interfaces:

  • VSCode: Visit localhost:8080 in your web browser. The password is yourpassword
  • QuestDB: Visit localhost:9000 in your web browser. The user/password are the default one: admin:admin
  • Grafana: Visit localhost:3000 in your web browser. The user/password are the default one: admin:admin
